### Form & Input Controls
- [Form](https://ui.shadcn.com/docs/components/form): Dynamic form layouts via React Hook Form and Zod.
- [Field](https://ui.shadcn.com/docs/components/field): Accessible input labels and helper error elements.

- [Textarea](https://ui.shadcn.com/docs/components/textarea): Multi-line text boxes.
- [Checkbox](https://ui.shadcn.com/docs/components/checkbox): Form checkbox selectors.
- [Radio Group](https://ui.shadcn.com/docs/components/radio-group): Mutual exclusion select lists.
- [Select](https://ui.shadcn.com/docs/components/select): Custom dropdown selectors.
- [Switch](https://ui.shadcn.com/docs/components/switch): Dynamic toggle switches.
- [Slider](https://ui.shadcn.com/docs/components/slider): Range slider inputs.
- [Calendar](https://ui.shadcn.com/docs/components/calendar): Rich calendar widgets.
- [Date Picker](https://ui.shadcn.com/docs/components/date-picker): Combined date selectors.
- [Combobox](https://ui.shadcn.com/docs/components/combobox): Searchable dropdown lists.
- [Label](https://ui.shadcn.com/docs/components/label): Accessible labels.
