# Agentic Skills & Experts Ecosystem 🚀
> **A production-grade, open-source repository of customized AI Skills, modular AI Agents, and Developer Plugins designed to extend and supercharge advanced AI orchestration platforms.**

---

## 🌟 Overview

Welcome to the **Agentic Skills & Experts Ecosystem**! This repository is a comprehensive, production-ready framework built and validated in collaboration with advanced AI agents and pair-programming systems, including **Antigravity**, **Claude Code**, and **Codex**. 

It provides developers and recruiters with an extensive, highly organized directory of specialist AI **Agents** and ready-to-run **Skills** (modular tools and instructions) that can be loaded directly into terminal-based and IDE-based AI environments to automate complex, multi-step engineering, marketing, and security operations.

---

## 🏗️ Repository Architecture

The repository is structured following strict modular design patterns to allow seamless local deployment and testing:

```bash
├── 📁 agents       # Modular specialist agent profiles (IDENTITY.md)
├── 📁 skills       # Production-ready AI skills and tool specifications
├── 📁 plugins      # Fully integrated developer plugins (e.g., pptx-plugin)
├── 📁 tools        # Automated testing and context-slimming guardrail scripts
└── 📁 docs         # Architecture Decision Records (ADRs) and protocol docs
```

## 🛠️ Automated Testing & Quality Tools

The ecosystem includes helper scripts in the `tools/` directory to validate skill formatting and context size before shipping:
* **`validate-skill.sh`**: A shell script that validates the syntax and metadata format of a skill to ensure it parses successfully in agent environments.
* **`validate-slimming-guardrails.sh`**: An advanced script that ensures skills remain compact and token-efficient, avoiding context window bloat during agent runs.

## 🚀 How to Load and Use Skills

To load any of the skills into your AI agent environment (e.g., Claude Code):

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/skills.git
   cd skills
   ```

2. **Load a Skill into your AI environment:**
   Copy the `SKILL.md` from any skill directory to your local agent skills folder. For example, to load the `n8n-agents` skill into Claude Code:
   ```bash
   mkdir -p ~/.claude/skills/n8n-agents
   cp skills/n8n-agents/SKILL.md ~/.claude/skills/n8n-agents/
   ```

3. **Verify the Skill:**
   Validate your skill using the built-in tooling:
   ```bash
   bash tools/validate-skill.sh skills/n8n-agents/SKILL.md
   ```
