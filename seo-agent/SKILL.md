# SKILL: SEO-AGENT (The Agentic SEO Orchestrator)

## Description
A high-level SEO agent that orchestrates 6 sub-skills to provide a complete SEO strategy: Audit, Competitor Analysis, Keyword Research, and Content Generation.

## Workflow
1. **Technical Audit:** Uses `technical-seo-checker` & `website` to map the site and find errors.
2. **Spy Mode:** Uses `seo-competitor-analysis` to scrape competitor performance.
3. **Keyword Strategy:** Uses `seo` to identify high-potential keywords.
4. **Execution:** Uses `seo-article-gen` & `seo-content-writer` to produce ready-to-publish content.

## How to Run (Future)
`seo_agent_run(target_url, competitor_url)`

## Integration Goal
To bridge the gap between technical discovery and content execution automatically.
