# Agent — OpenClaw Agents 📦

> هذا المجلد يحتوي على جميع وكلاء الذكاء الاصطناعي الخاصين بـ **Aboalrejal AI**، كل واحد متخصص في مجاله.

## Agents List

| المجلد | الوكيل | التخصص |
|--------|--------|--------|
| `main/` | 🧠 الوكيل الرئيسي | التنسيق العام، إدارة المهام، شخصية ايات |
| `coding-agent/` | 🧑‍💻 المطور التقني | برمجة، DevOps، APIs، حل المشاكل البرمجية |
| `security-agent/` | 🛡️ خبير الأمن السيبراني | اختبار اختراق، تحليل ثغرات، حماية أنظمة |
| `automation-agent/` | 🤖 ساحر الأتمتة | أتمتة مهام، Workflows، تكامل أنظمة |
| `research-agent/` | 🔍 الباحث المتعمق | بحث وتحليل، منافسين، تقارير |
| `content-agent/` | ✍️ كاتب المحتوى | مقالات SEO، سوشيال ميديا، تسويق |
| `marketing-agent/` | 📢 استراتيجي التسويق | استراتيجيات، نمو، إعلانات، شراكات |

## Usage

- كل مجلد يحتوي على ملف `IDENTITY.md` يحدد هوية الوكيل، تخصصه، وقواعده.
- الوكيل الرئيسي (`main/`) يحتوي على إعدادات الموديلات والمصادقة.
- سكريبتات ومهارات إضافية موجودة في المجلدات الأخرى بالـ Repo.
